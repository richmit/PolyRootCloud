Polynomial Root Cloud

Matlab code to plot the root clouds of parameterized polynomials.

I wrote this as part of a quick-n-dirty investigation into some ideas
related to polynomial root clouds.

The ultimate goal is to firm up some algorithmic aspects of the
problem, and implement it as an example for
[[https://github.com/richmit/mraster/][MRaster]].
