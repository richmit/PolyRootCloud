Polynomial Root Cloud

Matlab code to plot the root clouds of parameterized polynomials.

I wrote this as part of a quick-n-dirty investigation into some ideas related to polynomial root clouds. 

The ultimate goal was to firm up some algorithmic aspects of the problem, and implement a C++ version that I included in the examples collection of MRaster:
https://github.com/richmit/mraster/blob/master/examples/poly_root_cloud.cpp
